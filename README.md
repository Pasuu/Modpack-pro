# Modpack-pro
这里是方便Modpack网站添加和更新汉化的仓库


### 更新汉化只需要在issues中添加一个新的问题，问题名为 更新+整合包名字+版本 
### 添加汉化只需要在issues中添加一个新的问题，问题名为 添加+整合包名字+版本
### 内容填写云盘即可，后续我会添加进网站。

### 添加正在进行的汉化只需要在issues中添加一个新的问题，问题名为 正在进行+整合包名字+版本 （注，该问题不需要填写网盘链接）

### 当然你想提交PR也可以，具体方法就不在此细讲了。
